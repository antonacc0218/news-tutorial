# News Web Application

## Usage

### 1. Clone repository

```
git clone https://github.com/miniven/news-tutorial
```

### 2. Instal dependencies

```
npm install
```

or

```
yarn install
```

### 3. Run the App

```
npm run start
```

or

```
yarn start
```

App will be opened in browser at http://localhost:8080/

### 4. Build app for production


```
npm run build
```

or

```
yarn build
```
